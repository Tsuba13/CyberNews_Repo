Technologies Used

Backend

  • Python — the core language
  • Flask — web framework (routing, sessions, templates, request handling)
  • SQLAlchemy — ORM for database operations
  • Flask-Session — server-side filesystem sessions
  • Werkzeug — password hashing and WSGI utilities

Database

  • SQLite — local file-based relational database (users, articles)

Frontend

  • Jinja2 — templating engine (via Flask)
  • Vanilla JavaScript — client-side interactivity (greeting, article refresher, cookie notice)
  • CSS3 — custom styling, no framework

External APIs

  • NewsAPI — fetches live cybersecurity news articles

Infrastructure / Deployment

  • Render — hosting platform (web service + persistent disk)

Supporting Libraries

  • python-dotenv — environment variable management
  • urllib — HTTP requests to NewsAPI
